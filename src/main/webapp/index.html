Welcome to this Accela coding test

This web application allows you to manage persons and addresses.


  1. You can create a new person by giving a first name and a last name.
  2. You can view how many persons have been created: http://localhost:8080/persons
  3. You can count how many persons have been created: http://localhost:8080/count
  4. You can view and edit an existing person, or delete it. For example, to view a person with id #1, go to http://localhost:8080/persons/1
  5. You can also add an address to an existing person
  6. You can edit or delete an existing address
  7. You can view the addresses of an existing person. For example, to view the list of addresses of a person with id #1, go to http://localhost:8080/persons/1/addresses
Please use Postman to test all these different requests.
